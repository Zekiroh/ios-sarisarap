# SariSarap

iOS mobile application developed as an academic final project using Swift and SwiftUI.

The app digitizes food ordering for local eateries by allowing users to browse dishes, manage a cart, and complete a simple checkout flow.

---

## Project Overview  
SariSarap is a SwiftUI-based ordering application that includes:

- Splash screen entry
- Home screen with featured dishes
- Menu with searchable list
- Food detail view with quantity selector
- Cart management (add, update, remove)
- Receipt confirmation screen
- Light and Dark mode toggle

Each module demonstrates structured UI flow and state-driven updates using SwiftUI.

## Built With

- Swift  
- SwiftUI  
- NavigationStack  
- @StateObject / @EnvironmentObject  
- Custom Theme Manager
